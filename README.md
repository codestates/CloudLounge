# About

* 나의 현재 위치를 기반으로 주변 흡연구역을 나타내 줍니다.

* 흡연구역에 대한 별점 및 리뷰를 확인할 수 있고 직접 작성할 수 있습니다.

* 간편 신고 기능을 통하여 부정적인 고객 경험을 관리자에게 쉽게 전달하고 관리받을 수 있습니다.

    현재 버전은 서울특별시의 흡연구역만 지원합니다.
